# Queens

It is a challenging crown-placement logic puzzle game where you place queens on a grid of colored regions.

## How to play:
- Place exactly one queen in each row.
- Place exactly one queen in each column.
- Place exactly one queen in each colored region.
- Queens cannot touch each other, even diagonally (no adjacent placements).
